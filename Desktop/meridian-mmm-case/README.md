\## Математика и эконометрика



\### 2.1 Custom Adstock для TV

Реализована функция геометрического затухания: \*\*Aₜ = Xₜ + α·Aₜ₋₁\*\*



Исходя из графика α = 0.2 ближе к реальным данным



\### 2.2 Эластичность спроса по цене

\*\*Использованная формула:\*\* классическая точечная эластичность



$$E = \\frac{\\%\\ \\Delta Q}{\\%\\ \\Delta P} = \\frac{(Q\_2 - Q\_1)/Q\_1}{(P\_2 - P\_1)/P\_1}$$



\*\*Алгоритм расчета:\*\*

1\. Найдены все недели, где цена менялась (13 моментов)

2\. Оставлены только значительные изменения (>1%) — 9 случаев

3\. Для каждого изменения рассчитана эластичность по формуле выше

4\. В качестве итоговой оценки взята медиана (устойчива к выбросам)



\*\*Результаты:\*\*

\- Медианная эластичность: \*\*0.219\*\*

\- Интерпретация: спрос неэластичен — изменение цены слабо влияет на объем спроса

\- Это характерно для товаров с сильным брендом или лояльной аудиторией



\### 2.3 Bayesian Priors

Априорные вероятности помогают моделировать каналы с низкой активностью следующим образом:



1\. \*\*Стабилизация:\*\* Prior выступает регуляризатором, не давая модели делать необоснованные выводы на основе ограниченных данных

2\. \*\*Бизнес-ограничения:\*\* Priors можно задавать на измеримые показатели (ROI), а не абстрактные коэффициенты

3\. \*\*Управление уверенностью:\*\* Настройка стандартного отклонения позволяет балансировать между доверием к данным и экспертным мнением

4\. \*\*Консервативные сценарии:\*\* mROI priors помогают "прижать" оптимизационные рекомендации к историческим данным

\## Анализ эффективности каналов



\### Методика расчета

Для оценки эффективности каналов использовалась метрика \*\*GRP на доллар затрат\*\*:



$$ \\text{GRP per \\$} = \\frac{\\text{Total GRP}}{\\text{Total Spend}} $$



где:

\- \*\*Total GRP\*\* — суммарный охват по каналу за весь период

\- \*\*Total Spend\*\* — суммарные затраты на канал за весь период



Эта метрика показывает, сколько пунктов охвата приносит каждый потраченный доллар. Чем выше значение, тем эффективнее канал с точки зрения затрат на охват.
